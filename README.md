# 🎮 Video Game Sales Analysis

## 📌 Project Overview

This project performs an end-to-end exploratory data analysis (EDA) on the **Video Game Sales dataset (`vgsales.csv`)** to understand global sales patterns across different regions, genres, platforms, and publishers.

The main objective is to identify:

* Regional market differences
* Genre and platform trends
* Publisher dominance
* Sales evolution over time
---

## 🧹 Data Cleaning

The following preprocessing steps were applied:

* Removed rows with missing values in **Year**
* Filled missing **Publisher** values with `"Unknown"`
* Converted **Year** column into integer type
* Verified missing values after cleaning

---

## 📊 Exploratory Data Analysis

The analysis includes:

* Top 10 genres by global sales
* Top 10 platforms by global sales
* Top 10 publishers by global sales
* Best-selling games
* Regional sales comparisons
* Yearly global sales trend
* Decade-level performance analysis

---

## 🌍 Regional Market Analysis

A dedicated regional analysis was performed to compare:

* **North America**
* **Europe**
* **Japan**

### Key Findings

* **NA** contributes the highest share of global sales
* **EU** follows similar market trends to NA
* **JP** shows distinct genre preferences, especially RPG games

This highlights clear differences in regional gaming markets.
